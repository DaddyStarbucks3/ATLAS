# Gone Phishin' 🎣

Gone Phishin' is a Python terminal-based security awareness game that teaches users how to identify phishing emails.

## Overview

This application presents simulated email messages and prompts the user to classify each as:

- Phishing
- Legitimate

After each response, the game provides key indicators and an explanation to reinforce phishing detection skills.

## Features

- 20 prebuilt email scenarios
- Randomized question selection
- Score tracking
- Post-game review of missed questions
- Modular design (separated into data, logic, and UI)

## Technologies Used

- Python 3
- Modular project structure
- Basic OOP principles
- Structured input validation

## How to Run

From inside the project folder:

```bash
python3 main.py
```
